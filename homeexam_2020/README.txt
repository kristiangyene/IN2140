How to make executable files:

$ make all

or 

$ make server
$ make client



How to clean binaries:

$ make clean



How to run program:

- Have two terminals open.

Terminal 1:
$ make run_server

Terminal 2:
$ make run_client



How to run programs with valgrind:

- Have two terminals open.

Terminal 1:
$ make val_server

Terminal 2:
$ make val_client
